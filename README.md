# MobileNet-V4--pytorch

A PyTorch implementation of the MobileNetV4 Conv architecture based on the paper:

MobileNetV4: Universal Models for the Mobile Ecosystem

Implemented:
- ConvBNAct blocks
- Universal Inverted Bottleneck (UIB)
- MobileNetV4 Conv backbone
- Image classification head

Status:
- Educational implementation
- Not yet verified against official checkpoints
